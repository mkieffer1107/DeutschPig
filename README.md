# 🇩🇪 DeutschPig 🐷
DeutschPig is a version of the language [Pig](https://esolangs.org/wiki/Pig), that simulates a pig that is German.

#### Ersetzen!

```sh
python interpreter.py <filename>
```

#### Erstellen!
```sh
python interpreter.py 
Schnüffel 🐽> <filename>SCHWEIN<file-content>
```
